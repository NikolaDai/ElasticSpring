## Basic Knowledge
* about maven: https://spring.io/guides/gs/maven/

      Maven配置文件pom.xml中<scope>属性的取值及其含义：
      * compile(默认值)：依赖包在编译时需要就绪
      * Provided：编译时要求的依赖包尽在容器运行代码时需要就绪；
      * Test：编译和运行测试代码时依赖包就绪，运行时不需要；
